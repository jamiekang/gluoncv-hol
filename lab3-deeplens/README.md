# Lab 3 - DeepLens
**Abstract (in Korean)**

Lab 3에서는 앞의 lab에서 만든 object detection 모델을 `AWS DeepLens`에 deploy하고 실습해 봅니다.

- https://aws.amazon.com/deeplens/
